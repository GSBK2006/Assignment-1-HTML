# G S Bhavesh Kumar - Professional Portfolio

[Live Portfolio](https://g-s-bhavesh-kumar-profile.netlify.app/)


# Overview

This is a personal portfolio website built as part of my Week 1 HTML assignment. It serves as a professional landing page to showcase my academic background at NIT Puducherry, my internships, and my technical certifications. The project focuses on clean document structure and semantic web standards.

# Project Structure

The repository contains the following files:

```bash
index.html
profile.html
Bhavesh Kumar.jpg
Resume.pdf
```

index.html: The main portfolio page containing all sections (About, Experience, Education, Certifications, and Contact).

profile.html: A secondary page for detailed professional information.

Bhavesh Kumar.jpg: Profile image asset.

Resume.pdf: Professional resume for download.

Folder Structure

To ensure the website loads assets and links correctly, maintain the following directory layout:

/g-s-bhavesh-kumar-portfolio
│
├── index.html            # Main entry point
├── profile.html          # Detailed profile page
├── Bhavesh Kumar.jpg     # Profile photo
└── Resume.pdf            # Professional resume


Technical Features

Semantic HTML5: Used proper tags like <header>, <nav>, <main>, <section>, and <footer> for better accessibility and SEO.

Responsive Layout: Built with a mobile-first approach using CSS Flexbox and Grid to ensure compatibility across all devices.

Navigation: Implemented internal anchor links for smooth scrolling between sections.

Interactive Elements: Includes a functional contact form and a direct download button for my resume.

How to Run Locally

Clone the repository or download the ZIP file.

Open the folder in a code editor like VS Code.

Ensure all assets (image and PDF) are in the root directory.

Use the "Live Server" extension to launch index.html in your browser.
